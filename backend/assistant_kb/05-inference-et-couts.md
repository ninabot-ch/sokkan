# Inférence et coûts
- Deux modes (Profil → Modèle) : **BYOK** (votre clé Anthropic — vos prompts vont directement chez Anthropic, zéro intermédiaire) ou **inférence incluse** (modèles Qwen3 Coder servis depuis Francfort, prépayée par crédits, décomptée au token, contenu jamais conservé).
- Crédits : packs 25/100/500 CHF (+25 % de crédits dès 500). Solde visible dans Profil ; à solde épuisé l'inférence s'arrête (pas de découvert).
- L'onglet Coûts montre le coût estimé par session — utile pour repérer une session qui tourne pour rien. Bonnes pratiques : fermer les sessions terminées, une tâche = une carte = une session (pas une session fleuve).
- Les quotas journaliers en mode inclus sont des plafonds de protection, pas des limites d'usage normal.
